# Guess the Color Web Game

Test your memory! A color is shown for a few seconds. Memorize it, then use the color picker to match it as closely as possible. The closer your guess, the more points you earn. Reach a score of 100 to win!

## How to Play
- A color is displayed for 2.5 seconds. Memorize it!
- After it disappears, use the color picker to select the color you remember.
- Points are awarded based on how close your guess is to the original color:
	- Perfect match: +20
	- Very close: +10
	- Close: +5
	- Not close: 0
- The game ends when you reach 100 points.
- Click "Restart Game" to play again!

## How to Deploy to GitHub Pages
1. Create a new repository on GitHub (e.g., `guess-the-color-web`).
2. Push all files in this folder to your new repository.
3. In your repo settings, enable GitHub Pages (set source to `main` branch, `/root`).
4. Visit the provided URL to play your game online for free!
